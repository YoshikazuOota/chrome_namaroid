## 使い方
### 初期設定
- windows に nodeをインストール
- コマンドプロンプトを開く
- npm install を実行

### 使い方
- node server.js を実行
- NAMAROID クリップボックス版を開く
- NAMAROID のクリップボード監視機能を有効にする
- NAMAROID を開始
- chrome で http://localhost:3000/ を開く
- 表示されたページ内の開始ボタンを押す

## 使用パッケージをマニュアルインストール
npm install clipboard-polyfill
npm install copy-paste
npm install express
npm install socket.io
